# 🍿 PopcornMovies

PopcornMovies is a **React-based movie web app** that allows users to search for movies using a third-party API, view detailed movie information, rate movies, and maintain a personal "watched movies" list.

---

## Features

- 🔍 **Search Movies**: Search for any movie by title using the API.  
- 🎬 **View Movie Details**: See details like title, release date, runtime, genre, actors, director, plot, and poster.  
- ⭐ **Rate Movies**: Give your own rating to movies and track them.  
- ✅ **Watched Movies List**: Add movies to your personal watched list and view your ratings.  
- ⬅️ **Back Button**: Easily navigate back from movie details.  
- ⚡ **Responsive Design**: Works well on different screen sizes.  
- 🕹️ **Keyboard Support**: Close movie details using the Enter key.

## Technologies Used

- **React** - Frontend library for building user interfaces  
- **React Hooks** - `useState`, `useEffect` for state management and lifecycle  
- ** API** - Third-party API to fetch movie data  
- **CSS** - Styling the components  
- **Optional**: Local state for watched movies list
